Using `background-image` with a fixed number of repeats

Use cases? Photo Corners Decorated Frame Star Rating Notes

Photo Corners

Create a div that will act as a frame around two child elements:

  • An img element to contain the photo
  • A div>; to hold a photo corner background-image in each corner

To get the frame div to shrink-wrap around the img, use display: inline-block. To get the div that holds the corners to fit the frame div exactly, set its width and height to 100% of its parent's dimensions.

Set the position of thediv that holds the corners to absolute, and set its top and left properties to 0, so that it fits the parent div exactly.

Set the position property of the frame div to relative, instead of the default value of static. If you don't the corners will fit into the body element, by default.

Decorated Frame

Home sweet home!

Create a square <div> for the frame, and place a <p> element inside it. Use the same background-image four times.

Set the background-size of all instances of the background-image a value which represents an integer fraction of the dimensions of the parent <div>. For example: 20%, which would make each image be shown 5 times along one edge of the frame <div>. (You only need to provide this value once.)

Give two of the instances of the background-image a value for background-repeat of repeat-x and give the other two a background-repeat of repeat-y. Set the background-position of each instance of the image to top, right, bottom and left, to match the repeat directions.

Centre the text in the <p> element, and give it sufficient padding and an appropriate font-size, so that the text does not overflow into the frame area.
